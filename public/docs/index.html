Energietarife Österreich API Dokumentation

Visualisierung API Docs

Überblick

Kostenlose REST API für österreichische Strom- und Gastarife. Datenquelle ist der E-Control Tarifkalkulator. Alle Endpunkte liefern JSON, unterstützen CORS und sind öffentlich zugänglich.

Base URL: https://energietarife.vercel.app/api/v1

Endpunkte

GET /api/v1/tariffs
Historische Tarifdaten aller Anbieter (Energiepreis, Grundgebühr, Gültigkeitszeitraum)

Query-Parameter

ParameterTypBeschreibung
energy_typestringPOWER oder GAS optional
brandstringFiltert nach Anbieter-Name (Teilsuche) optional
fromdateStartdatum YYYY-MM-DD optional
todateEnddatum YYYY-MM-DD optional
min_pricenumberMindest-Energiepreis in ct/kWh optional
max_pricenumberMaximal-Energiepreis in ct/kWh optional
limitintegerAnzahl Ergebnisse default: 1000, max: 5000
offsetintegerErgebnisse überspringen default: 0

Response-Felder

FeldTypBeschreibung
product_namestringProduktbezeichnung
brand_namestringAnbieter/Marke
energy_typestringPOWER oder GAS
product_validity_fromdateGültig ab (YYYY-MM-DD)
energy_rate_ct_kwhnumberEnergiepreis in Cent/kWh (netto)
energy_rate_high_ct_kwhnumber?HT-Preis (bei Doppeltarif)
energy_rate_low_ct_kwhnumber?NT-Preis (bei Doppeltarif)
base_rate_centsnumber?Grundgebühr in Cent/Jahr
rate_typestringz.B. STANDARD
locationsstring?Verfügbare Regionen

Anwendungsbeispiele

Energieversorger — Wettbewerbsanalyse

Vergleichen Sie die eigenen Tarifentwicklungen mit dem Markt. Filtern Sie nach Energieart und Zeitraum, um Preistrends und Positionierung zu analysieren.

GET /api/v1/tariffs?energy_type=POWER&from=2024-01-01&to=2025-01-01
Softwarehersteller — Tarifvergleich integrieren

Nutzen Sie die API, um aktuelle und historische Tarife in Ihre Vergleichsportale, Apps oder Beratungstools einzubinden.

GET /api/v1/current?energy_type=POWER&limit=50
Data Science — Preisanalyse

Laden Sie den gesamten historischen Datensatz für Analysen, Forecasting oder Reporting.

GET /api/v1/tariffs?limit=5000
# Oder direkt als JSON-Datei:
GET /data/historical.json
Monitoring — Preisänderungen erkennen

Überwachen Sie einen bestimmten Anbieter und erkennen Sie neue Tarifprodukte oder Preisanpassungen.

GET /api/v1/tariffs?brand=wien%20energie&from=2025-01-01

Hinweise

Rate Limiting: Kein Rate Limit. Responses werden 1h gecached (CDN).

CORS: Alle Origins erlaubt — direkter Aufruf aus dem Browser möglich.

Rohdaten: Die JSON-Dateien sind auch direkt unter /data/historical.json, /data/current.json, /data/brands.json und /data/stats.json abrufbar.

Datenquelle: E-Control Tarifkalkulator. Daten werden periodisch aktualisiert.

Lizenz: Open Data. Quellenangabe erwünscht.
